# User Authentication using JWT

## Description
This project demonstrates user authentication using JWT, MongoDB, Express.js, and Node.js.

## Features
- User Registration
- User Login
- Password Encryption
- JWT Authentication

## Technologies Used
- Node.js
- Express.js
- MongoDB
- JWT
- bcryptjs

## Installation
npm init -y
npm install express mongoose bcryptjs jsonwebtoken body-parser

## Run
node server.js
